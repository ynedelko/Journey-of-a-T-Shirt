# _Journey of a T-Shirt_

##### An educational app that allows you to follow the journey of a T-shirt.

#### By _**Yelena Nedelko**_

## Description

_Users:_
* _Will learn where T-shirts are made, who makes them, and where they end up._
* _Will have the opportunity to explore facts about 10 countries with the highest apparel imports to the US._
* _Will be prompted to input data about where their t-shirt was made and see the results tallied with other users._

_View App:_ to be added to Heroku

## Database Setup/ Installation

_In rake:_<br>
_db:setup, (The seeded data will be loaded with db:setup.)_<br>
_all database tables listed in schema.rb_
<br>

_Webserver testing:_
* _Install required gems_
* _$ bundle install_
* _rake db:test:prepare_
* _$ rails server_
* _Navigate to localhost:3000_
